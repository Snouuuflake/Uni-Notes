\begin{defbox}{Ecuación diferencial (ED)}
  Una ecuación que contiene las derivadas de una o más funciones desconicidas (o variables dependientes)
  \begin{center}
    \[
      \frac{dy}{dx} = 2x 
      \to
      \int dy ~dx = \int 2xdx
    \]
    \[
      \text{Conjunto de solución general: } y = 2x^{2} + C
    \]
  \end{cener}
\end{defbox}
